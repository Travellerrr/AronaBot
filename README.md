# 机器人阿洛娜额外功能插件
## 指令
目前共三个指令

| 指令 | 功能 | 作用域 |
| :------------: | :------------: | :------------: |
| `#原神角色列表` | 查看最新版本原神角色列表 | **全部** |
| `#原神攻略 [角色]` | 查看某一原神角色攻略 | **全部** |
| `#今日运势` | 查看今日运势 | **QQ群** |
| `#监控` | 查看服务器资源占用 | **QQ群** |

`#监控`指令需要在`config/cn.travellerr.AronaBot/Config.yml`下设置主人
## 安装方式
直接将仓库下载至本地，自行构建后放入mirai的plugin文件夹即可

`Version = `1.0.0`
